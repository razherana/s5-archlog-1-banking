Banking Courant API Documentation

RESTful API for managing current accounts (compte courants) and transactions.

Base URL: http://localhost:8080/api

Account Management (Compte Courant)

GET
/api/comptes
Get all accounts
GET
/api/comptes/{id}
Get account by ID
GET
/api/comptes/user/{userId}
Get all accounts for a user
POST
/api/comptes/user/{userId}
Create new account for user
DELETE
/api/comptes/{id}
Delete account

Transaction Management

GET
/api/transactions
Get all transactions
GET
/api/transactions/{id}
Get transaction by ID
GET
/api/transactions/compte/{compteId}
Get all transactions for an account
POST
/api/transactions/depot
Make a deposit
Request Body: 
{
  "compteId": 1,
  "montant": 100.00,
  "description": "Salary deposit"
}
POST
/api/transactions/retrait
Make a withdrawal
Request Body: 
{
  "compteId": 1,
  "montant": 50.00,
  "description": "ATM withdrawal"
}
POST
/api/transactions/transfert
Transfer money between accounts
Request Body: 
{
  "compteSourceId": 1,
  "compteDestinationId": 2,
  "montant": 25.00,
  "description": "Transfer to friend"
}

Response Format

All API responses use JSON format. Successful responses return the requested data or confirmation messages. Error responses include HTTP status codes and error messages.

Example Success Response:

{
  "id": 1,
  "name": "John Doe",
  "email": "john.doe@example.com"
}

Example Error Response:

{
  "error": "User not found"
}

HTTP Status Codes

  • 200 OK: Request successful
  • 201 Created: Resource created successfully
  • 204 No Content: Resource deleted successfully
  • 400 Bad Request: Invalid request data
  • 404 Not Found: Resource not found
  • 500 Internal Server Error: Server error
